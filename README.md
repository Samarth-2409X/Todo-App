# 🌈 Colorful Todo App

A simple and visually appealing Todo List web app built using **HTML, CSS, and JavaScript**.

## ✨ Features

* ➕ Add new todos
* ❌ Delete individual todos
* 🗑️ Delete first todo
* 🗑️ Delete last todo
* 🎨 Random colorful backgrounds for each task
* ⚡ Smooth UI with animations

## 🛠️ Tech Stack

* HTML
* CSS (Gradients & Animations)
* Vanilla JavaScript (DOM Manipulation)

## 🚀 How to Run

1. Download or clone the repository
2. Open the `index.html` file in your browser
3. Start adding your tasks!

## 📸 Preview

A clean and colorful interface that makes managing tasks fun and simple.
